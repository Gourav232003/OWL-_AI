# 🦉 OWL AI

An interactive and simple AI chatbot web application built using Flask or Streamlit.  
OWL AI is designed to provide intelligent responses, assist with programming queries, and deliver a clean, user-friendly chat experience.

---

## 🚀 Features

- 💬 Interactive chat interface
- 🧠 Predefined intents & smart responses
- ⏰ Dynamic time & date responses
- 📚 Programming help support
- 🎯 Simple and clean UI
- 🌐 Web-based application

---

## 🛠 Tech Stack

- Python
- Flask / Streamlit
- HTML
- CSS
- Git & GitHub

---

## 🎯 Supported Intents

- Greeting (Hello, Hi)
- About Owl AI
- Help Commands
- Programming Questions
- Motivation Quotes
- Time & Date
- Goodbye / Exit

## ▶️ How to Run the Project

### 1️⃣ Clone the Repository

git clone https://github.com/Gourav232003/OWL-_AI.git  
cd OWL-_AI  

---

### 2️⃣ Install Requirements

If using Flask:

pip install flask  

If using Streamlit:

pip install streamlit  

---

### 3️⃣ Run the Application

For Flask:

python app.py  

For Streamlit:

streamlit run app.py
